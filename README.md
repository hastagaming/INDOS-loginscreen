# 󰣇  INDOS

**INDOS** is a sophisticated Terminal User Interface (TUI) workspace for Android (Termux) and Linux, heavily inspired by the **Hyprland** and **Waybar** aesthetics.

## 󱐋 Features
- **Dynamic Session Management**: Add multiple terminal panels with a single click.
- **NerdFont Integration**: High-quality icons for a modern look.
- **Auto-User Detection**: No hardcoded names; it adapts to your system.
- **Clean Architecture**: Modular Python code with external CSS styling.

## 󰏔 Requirements
- Python 3.x
- A Terminal with **NerdFont** support (e.g., Termux with a custom font).
